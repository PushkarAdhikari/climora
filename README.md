# 🌤️ Climora: Weather Forecast

A modern, beautifully designed weather application built with **Expo
React Native**. Climora delivers accurate weather forecasts through an
elegant Aurora-inspired interface with smooth animations and a premium
user experience.

------------------------------------------------------------------------

## ✨ Features

-   📍 Current weather based on your location
-   🔍 Search weather for cities worldwide
-   ⏰ Hourly weather forecast
-   📅 7-day weather forecast
-   🌡️ Feels-like temperature
-   💧 Humidity
-   🌬️ Wind speed
-   🌧️ Rain probability
-   🌞 UV Index
-   👁️ Visibility
-   🌅 Sunrise & Sunset
-   🌙 Dark mode support
-   🎨 Aurora-inspired modern UI

## 🚀 Getting Started

### Clone the repository

``` bash
git clone <repository-url>
cd climora
```

### Install dependencies

``` bash
npm install
```

### Start the development server

``` bash
npx expo start
```

### Build for Android

``` bash
eas build -p android --profile production
```

## 🔒 Permissions

Climora requests:

-   ACCESS_FINE_LOCATION
-   ACCESS_COARSE_LOCATION

These permissions are used only to provide accurate local weather
forecasts.
